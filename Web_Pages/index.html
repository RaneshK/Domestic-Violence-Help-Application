How To Use Rosepetals

Download the Application

Rosepetals is available for FREE
on IOS and Android app stores
Follow the link to download it right now!

Sign Up or Log In

Once the user has signed up or logged in, the mobile interface of the application will randomly generate questions on
different topics for which there will be three answers with different background colors to choose from

Start Playing

Once the user answers a question, a prompt will appear to notify them
of the succcessful submission of the option chosen
DISCLAIMER:Users answers are recorded in our secure database and Rosepetals team will not be engaging in selling user data
like many other popular applications!
